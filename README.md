## Introduction
使用 Vue 實作手錶購物平台功能，串接後端 API 所提供資訊。

## Features
- 前台
  - 產品 Banner
  - 產品促銷分類
  - 產品評分效果
  - 獲取優惠折扣碼
  - 產品圖片瀏覽
  - 產品分類列表 (網格模式、清單模式)
  - 產品詳細頁面
  - 相關推薦產品
  - 購物車
  - 願望清單
  - 後臺登入 (記住帳密)
  - RWD 響應式

- 後台
  - 產品管理
  - 訂單管理
  - 優惠券管理
  - 簡易模式選單
  - 分頁選單
  - RWD 響應式
